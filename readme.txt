== Description ==

Places a paywall after every <code><!--more--></code> into your post(s).

== Installation ==

1. Upload `startspot-wp-paywall.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter a `Kenfmcoin address` into the plugin's admin menu where payments should be sent to.
4. Enter a `deposit factor` (>0, <5000000000) into the plugin's admin menu that determines the costs to view your posts.

== Frequently Asked Questions ==

= Where to get a Kenfmcoin address? =

From your Kenfmcoin paper/mobile/desktop wallet

= What is the best deposit factor? =

Costs/Hour in Kenfmcoin = (3600/(deposit factor))
